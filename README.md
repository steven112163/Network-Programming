# Network-Programming
Network Programming Spring 2020 NCTU

## Prerequisites
* Python >= 3.6.9
* SQLite3 >= 3.22.0
* termcolor >= 1.1.0
* boto3 >= 1.13.8

## Submodule
Test cases from https://git.cs.nctu.edu.tw/thlin56020/nctu_nphw3_demo.git for hw3. After cloning, run following commands:
```shell script
git submodule init
git submodule update
```
